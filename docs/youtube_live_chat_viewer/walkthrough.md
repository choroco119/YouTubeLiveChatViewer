# YouTubeLiveChatViewer 開発ウォークスルー

## 実装された機能の詳細

### 1. 高機能読み上げフィルタ
単なる読み上げだけでなく、配信の「聞きやすさ」を重視したフィルタを実装しました。
- **NGワード置換**: 暴言などを直接読まずに「不適切な表現」等に置き換えて読み上げることで、放送事故を防ぎます。
- **長文省略**: 指定文字数を超えたコメントを「以下略」で端折ることで、読み上げの停滞を防ぎます。

### 2. リスナーとの交流を深めるメモ機能
リスナーを「ユーザーID」で一意に識別し、配信者が個別のメモを記録できます。
- 以前の配信で何を話したか、どんな属性のリスナーかを記録し、次回来訪時にすぐ確認できます。
- 専用の「ユーザー管理ウィンドウ」で、配信後でもゆっくりメモの整理が可能です。

### 3. 効果音（SE）による演出
特定のコメントに反応して音を鳴らすことができます。
- キーワードごとに音量設定が可能。
- 「わこつ」での挨拶音や、特定のネタキーワードでのリアクション音など。

### 4. 堅牢なログ管理
配信の全記録を確実に保存します。
- **ID付きログ**: ユーザー名だけでなく不変のチャンネルIDを記録するため、名前が変わっても同一人物として識別可能です。
- **過去ログ閲覧**: 保存されたログファイルを読み込み、当時の盛り上がりをコメントログ欄に再現できます。

## 技術的解決

- **COM連携**: CeVIO AI との直接連携により、低遅延で自然な読み上げを実現。
- **非同期アーキテクチャ**: `tokio` によるマルチスレッド処理で、コメントの取得、UIの描画、読み上げ、SE再生が互いに干渉せずスムーズに動作します。
- **Gridレイアウト**: `egui` の `Grid` を駆使し、多数の設定項目を揃えて表示する工夫を行いました。

## ビルド・検証結果
- `cargo build --release` による最適化済みバイナリを生成。
- 日本語入力、日本語読み上げ、ファイル選択ダイアログ、音声再生（rodio）のすべてが正常に動作することを確認済み。
