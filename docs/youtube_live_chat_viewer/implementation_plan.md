# YouTubeLiveChatViewer 実装計画（最終まとめ）

## プロジェクト概要
YouTubeライブのコメントを監視し、CeVIO AIで読み上げる機能を備えた高機能コメントビューア。
配信の記録、リスナー管理、NGワード対策、効果音演出など、配信者の活動をトータルでサポートすることを目的とする。

## 主要コンポーネント

### 1. YouTube 連携 (`youtube.rs`)
- 指定されたURL/動画IDからライブチャットを継続的にポーリング。
- 動画タイトルをwatchページから正確に取得。
- モデレーター、メンバー、配信者などの属性を識別。

### 2. 音声合成エンジン (`cevio.rs`)
- CeVIO AI の COM インターフェースを直接制御。
- キャスト選択、感情パラメータ、音声パラメータ（速度・ピッチ等）のリアルタイム反映。
- 非同期キューによる円滑な読み上げ処理。

### 3. テキスト処理 (`text_filter.rs`, `dictionary.rs`)
- 正規表現ベースのユーザー辞書。
- URL除去、草(www)の変換、連続記号の整理。
- 文字数制限（切り捨て ＋ 「以下略」付与）の動的処理。

### 4. 永続化と管理 (`settings.rs`, `app.rs`)
- 設定ファイルの自動保存（JSON形式）。
- ユーザーID（チャンネルID）に紐付いた永続的なメモ機能。
- ログの自動保存（タイムスタンプ・ID・名前・本文）。
- 過去ログのインポートと閲覧機能。

### 5. セキュリティ・演出 (`app.rs`)
- NGユーザー：特定のユーザーからのコメント読み上げを完全スキップ。
- NGワード：特定のワードを含むコメントを「不適切な表現」等の定型文に置換して読み上げ。
- 効果音（SE）：特定キーワードに反応してオーディオファイル（mp3/wav等）を再生。

## UI/UX設計
- `egui` による高レスポンスなダークテーマUI。
- 左側に設定パネル、右側に巨大なコメントログを配置。
- ユーザー管理、辞書登録、NG設定、SE設定を独立したセクションやウィンドウで提供。
- 日本語フォント（M+ FONT）の埋め込みによる文字化け防止。

## 検証済み機能
- Windows 10/11 環境での CeVIO AI 連携。
- 長時間配信でのメモリ・CPU使用率の安定。
- 数千件規模のコメントログの保存と読み込み。
