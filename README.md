# YouTubeLiveChatViewer

![YouTubeLiveChatViewer Icon](assets/icon.png)

YouTubeライブのコメントをリアルタイムで取得し、CeVIO AI で読み上げる高機能コメントビューアです。
配信者の利便性を追求し、リスナー管理や読み上げフィルタ、効果音演出などの基本機能に加え、Gemini APIによる自動返答生成やOBS配信用オーバーレイ画面の配信機能などを凝縮しました。

## ✨ 主な機能

### 🗣️ CeVIO AI 連携
- **リアルタイム読み上げ**: 配信中のコメントを遅滞なく読み上げます。
- **柔軟な音声設定**: キャストの選択はもちろん、速度、ピッチ、音量、さらに感情パラメータ（哀しみ、怒り、嬉しい、落ち着き等）をアプリ内で直接調整可能です。

### 🤖 Gemini AI 連携
- **自動要約・返答生成**: 一定間隔（または新規コメント検知時）で、リスナーコメントに基づいた自動返答や要約を生成し、CeVIO AIで読み上げさせることが可能です。
- **コンテキストの自動付与**: 接続時に取得した「配信タイトル」をプロンプトの先頭に自動付加し、配信テーマに沿った的確な対話を生成します。
- **APIエラー対策**: `429 Too Many Requests` 回避のため、指数バックオフ付きリトライ処理（最大3回）を内蔵しています。
- **モデル選択**: `gemini-2.5-flash` などの最新主要FlashモデルをUI上で簡単に選択でき、カスタムモデル名の手動入力にも対応しています。

### 📺 OBS配信用オーバーレイ（内蔵Webサーバー）
- **超軽量内蔵サーバー**: アプリ内で軽量なHTTPサーバー（デフォルトポート: `3000`）が稼働し、OBSブラウザソース向けの透過画面を配信します。
- **統合上下レイアウト**: 上側にGeminiの回答（吹き出し形式、フェード＆スライドアニメーション付き）、下側に通常コメント（下から上へスクロール追従するチャットフィード）を表示します。
- **動的非表示制御**: アプリ側でGemini連携がオフの時は、AIの回答表示枠が自動的に非表示となり、通常コメントのみが画面全体に広がるように自動でレイアウトが最適化されます。
- **ユーザー役割の可視化**: 配信者（👑 金）、モデレーター（🔧 水色）、メンバー（🟢 緑）などの役割に応じたアイコンバッジと名前の色分けに対応しています。

### 📝 リスナーメモ管理
- **ユーザーID紐付け**: チャンネルIDごとにメモを保存。リスナーが名前を変えてもメモが引き継がれます。
- **以前の会話を記録**: 「以前こんな話をした」「初見さん」などの情報を記録し、次回来訪時にすぐ確認できます。
- **データ分離**: メモは `user_notes.json` に保存され、設定ファイルとは別に管理できます。

### 🛡️ 高度な読み上げフィルタ
- **NGワード置換**: 指定した単語を「不適切な表現」などに置き換えて読み上げ。放送事故を未然に防ぎます。
- **NGユーザー**: 特定ユーザーの読み上げを完全にスキップ。
- **長文短縮**: 設定した文字数を超えるコメントは「以下略」などを付けて自動的に短縮。

### 🎵 効果音（SE）演出
- **キーワード反応**: 特定の単語に反応して、指定の音声ファイル（mp3/wav等）を再生。
- **個別音量設定**: SEごとに音量を調整でき、配信の演出を彩ります。

### 📊 ログ管理
- **自動保存**: 配信中の全コメントをタイムスタンプ、ユーザーID、ユーザー名、本文と共に自動記録（`logs/` ディレクトリ配下にテキスト保存）。
- **過去ログ閲覧**: 保存されたログファイルを読み込み、当時のコメントをアプリ上で再現・振り返ることができます。

### ⚙️ すっきりとしたUIデザイン（タブ化）
- **カテゴリタブ切り替え**: 設定項目を「音声」「AI」「配信」「フィルタ」「効果音」「その他」にタブ化して整理し、縦長スクロールを不要にしました。

## 🛠️ 動作要件

- **OS**: Windows 10 / 11 (64bit)
- **必須ソフトウェア**: [CeVIO AI](https://cevio.jp/) がインストールされ、利用可能なキャストがあること。

## 📦 インストールと使い方

1. 本アプリケーション（`youtube-live-chat-viewer.exe`）をダウンロードし、任意のフォルダに配置します。
2. `youtube-live-chat-viewer.exe` を実行します。
3. YouTubeのライブ配信URLまたは動画IDを入力し、「接続」ボタンを押してください。
4. 左側の設定タブから、好みに合わせて音声、Gemini、OBS配信などの設定を行えます。

## 💾 データファイルについて

アプリケーションを実行すると、同じフォルダに以下のファイルが作成されます：
- `settings.json`: アプリの設定（音声パラメータ、NG設定、SE設定など）。
- `user_notes.json`: リスナーごとの個別メモ。
- `logs/`: 配信ごとのコメントログが保存されるディレクトリ。

## ⚠️ 注意事項

- 本ソフトは CeVIO AI の COM インターフェースを利用します。ライセンス認証が済んでいる必要があります。
- YouTubeの仕様変更により、コメント取得ができなくなる場合があります。
